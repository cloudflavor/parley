Quickstart

Get a review running locally in less than five minutes.

1. Build

cargo build --release

2. Create a review

./target/release/parley review create my-review
./target/release/parley review start my-review

3. Open the TUI

./target/release/parley tui --review my-review

4. Core keys

  • h/l file prev/next
  • j/k line down/up
  • m or c create thread
  • r reply to thread
  • a/o set thread addressed/open
  • ? open in-app shortcuts reference

5. Refresh after code changes

R

Use R inside the TUI to reload review metadata and git diff.
